# バイト代 計算アプリ

バイトで「月初めから今日までにいくら稼いだか」を記録・集計するシンプルな Web アプリです。
給料が月ごと支払いのため、月が替わると今月分の合計が自動でリセットされ、過去の月は履歴として残ります。

## 公開ページ

https://ksho111213-droid.github.io/baito-app/

## 使い方

1. 日付・働いた時間・時給・交通費を入力して「この勤務を追加」
2. 一番上に「今月の合計(月初め〜今日)」が表示される
3. 月が替わると今月分は自動でゼロから再集計。前の月は「過去の月」に残る
4. 「設定」で交通費の既定額を変更できる

## 仕組み

- HTML / CSS / バニラ JavaScript のみ。ビルド不要・オフライン動作。
- 入力データは各端末のブラウザ内（localStorage）にのみ保存され、外部には送信されません。
  そのため他人には見えませんが、別の端末とは同期しません。
