# バイト代 計算アプリ

バイトで「月初めから今日までにいくら稼いだか」を記録・集計するシンプルな Web アプリです。
給料が月ごと支払いのため、月が替わると今月分の合計が自動でリセットされ、過去の月は履歴として残ります。

## 公開ページ

https://ksho111213-droid.github.io/baito-app/

## 使い方

1. **給与の種類**(時給 / 日給)と **入力方法**(時刻で入力 / 時間で入力)を選ぶ
2. 日付・勤務時間・金額・交通費を入力して「この勤務を追加」
   - 「時刻で入力」…… 開始・終了・休憩を**回転ホイール**で選ぶと、金額を自動計算
   - 「時間で入力」…… 働いた時間(◯時間◯分)を直接ホイールで選ぶ
3. 一番上に「今月の合計(月初め〜今日)」が表示される
4. 各記録の ✎ で内容を修正、× で削除できる
5. 月が替わると今月分は自動でゼロから再集計。前の月は「過去の月」に残る
6. 「設定」で交通費・休憩の既定値を変更できる

## 主な機能

- **時給 / 日給**の切り替え(日給は固定額。時間は任意)
- **時刻から自動計算**：開始〜終了と休憩から実働時間・金額を算出。プレビュー表示
- **夜勤対応**：終了が開始より前なら翌日として計算(例 22:00〜翌6:00 = 8時間)
- **15分刻みの回転ホイール**：スマホでは指で回して選べる(Googleマップ風)
- **記録の編集**：追加済みの勤務を後から修正
- **月ごとの集計と履歴**：今月の合計、過去の月の折りたたみ表示

## 仕組み

- HTML / CSS / バニラ JavaScript のみ。ビルド不要・オフライン動作。
- 入力データは各端末のブラウザ内（localStorage）にのみ保存され、外部には送信されません。
  そのため他人には見えませんが、別の端末とは同期しません。

## ファイル

| ファイル | 役割 |
|---|---|
| `index.html` | 画面の骨組み・サービスワーカー登録 |
| `style.css` | 見た目（濃色 + ゴールドのダークテーマ） |
| `app.js` | 記録・計算・表示のロジック（構成とデータ構造は冒頭コメント参照） |
| `sw.js` | オフライン起動用のサービスワーカー（更新時は `CACHE_VERSION` を上げる） |
| `manifest.json` | PWA（ホーム画面追加）の設定 |

## 開発メモ

- `app.js` の先頭コメントに「ファイルの構成」と「記録1件(entry)の形」をまとめてあります。まず先に目を通すと編集しやすいです。
- ファイルを更新して公開ページに反映するには、`sw.js` の `CACHE_VERSION` を1つ上げてから commit → push します（キャッシュを確実に切り替えるため）。
